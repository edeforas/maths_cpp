This package contains the source and executables for compressing images using wavelets, however, it is just for educational purpose and has its own format and viewer. It is based on the "Licence EEA" (Electronique, Electrotechnique et Automatique) end of year's project .

It is written in C, compiling under Dos ( Borland C3.1) and runs in VGA Mode.

Included is a little Mandelbrot set viewer to make sample images.
